## Requirements

* php v7.4 ( [XAMPP](https://www.apachefriends.org/download.html) )
* [Node.js](https://nodejs.org/) v14 or higher
* [Composer](https://getcomposer.org/)

## Installation

* cd assets/admin
* npm install
* cd ../..
